# 示例：验证概率不等式
import numpy as np
from scipy import stats

def verify_hoeffding(n=10000):
    # Monte Carlo 验证 Hoeffding 不等式
    X = np.random.uniform(-1, 1, n)
    delta = 0.1
    P = np.mean(np.abs(X) > delta)
    bound = 2 * np.exp(-2 * n * delta**2)
    return P, bound

CLAUDE.md 内容

# CLAUDE.md

## 重要规则
1. **"记住" 规则**: 用户说"记住 XXX"时，必须写入 CLAUDE.md
2. **LaTeX 格式严格性**: 禁止 Markdown 格式，必须使用纯 LaTeX
3. **先读原文再写笔记**: 禁止凭想象编写数学证明
4. **作业路径**: Google Drive 相应目录
5. **Unicode 禁止规则**: 禁止 n₁ 等 unicode 下标，必须用 $n_1$

## 数学符号习惯
- 概率: $\mathbb{P}(A)$
- 期望: $\mathbb{E}X$
- 方差: $\text{var}$
- 协方差: $\text{cov}$
- 独立性: $A \Perp B$

## 写作风格
- 动机优先: 每个概念引入前先解释"为什么需要它"
- 历史脉络: 注重概念的起源和发展历史
- 叙事流畅: Stein《傅里叶分析》风格
