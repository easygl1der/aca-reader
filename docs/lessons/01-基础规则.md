# 经验教训汇总

## 格式规范

- **LaTeX 禁止 Markdown**：禁止使用 `-`、`1.`、`**`、`*`、``` 等 Markdown 语法
- **Label 格式**：`\label{def:名称}`，引用用 `\cref{名称}`

## 阅读技巧

- **先读目录**：读取任何 PDF 书籍时，必须先读 Table of Contents 确认章节结构
- **对照原文**：直接读取原文对照，避免凭记忆写笔记

## 技术细节

- **R 绘图**：默认不支持中文，所有 R 图表必须使用英文标签
- **LaTeX 图片**：[H] 强制固定图片位置（需 float 宏包）
- **论文链接**：1998年以前的论文通常没有 arXiv 版本，应使用 DOI

## 项目结构

- **Memory 截断**：MEMORY.md 超过 200 行会被截断，重要内容写入 CLAUDE.md
- **Claude.md 精简**：详细规范拆分到 docs/ 目录，CLAUDE.md 保持简洁
