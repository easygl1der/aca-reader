Curves

微积分研究的是函数的变化，而微分几何研究的是空间中的曲线和曲面。本章我们学习曲
线的理论，为后续学习曲面奠定基础。


1.1   Introduction
本章我们将学习空间曲线和平面曲线的基本理论。曲线是微分几何中最基本的对象，我们
可以把它理解为空间中一条连续运动质点的轨迹。
  在微分几何中，我们关心的是曲线的几何性质——如弯曲程度、扭曲程度——而不是
它如何被参数化。这引导我们引出弧长、曲率、挠率等核心概念。


1.2   Parametrized Curves
定义 1.1 (参数曲线). 设 I ⊂ R 是一个区间。映射

                                 α : I → R3

称为一条参数曲线（parametrized curve）
                           。如果 α 是可微的，我们称其为可微曲线（differ-
entiable curve）。

定义 1.2 (正则曲线). 如果对于所有 t ∈ I，导数 α′ (t) ̸= 0，则称 α 为正则曲线（regular
curve）。这意味着曲线在每一点都有切向量，且切向量不为零。

注 1.1. 正则性条件保证曲线是光滑的，没有尖点或结点。

定义 1.3 (简单曲线). 如果映射 α 是单射（即 α(t1 ) = α(t2 ) 蕴含 t1 = t2 ），则称曲线为简
单曲线（simple curve）。

例 1.1 (圆). 参数方程
                  α(t) = (a cos t, a sin t, 0),   t ∈ [0, 2π]
表示 R3 中半径为 a 的圆。这是平面曲线（所有点都在 xy-平面上）。

例 1.2 (圆柱螺旋线). 参数方程

                           α(t) = (a cos t, a sin t, bt),       t∈R

曲线的区别在于是否存在挠率：平面曲线的挠率恒为零。

1.4    The Vector Product in R3
向量积（叉积）是研究空间曲线的重要工具。

积（cross product）定义为

                   u × v = (u2 v3 − u3 v2 , u3 v1 − u1 v3 , u1 v2 − u2 v1 ).

1.5     The Local Theory of Curves Parametrized by Arc
        Length
这是曲线理论的核心部分。我们将看到曲线的形状完全由两个函数决定：曲率（curvature）
和挠率（torsion）。

1.5.1   The Curvature
定义 1.10 (曲率). 设 α(s) 是单位速度曲线。定义曲率（curvature）为
                                         k(s) = ∥α′′ (s)∥.
即加速度向量的模长。
注 1.6. 曲率描述曲线弯曲的程度。直线（曲率为 0）完全不弯曲，圆（曲率为常数 1/a）
弯曲程度均匀。
例 1.3 (圆的曲率). 单位圆的参数方程 α(s) = (cos s, sin s, 0)，则
                  α′ (s) = (− sin s, cos s, 0),     α′′ (s) = (− cos s, − sin s, 0),
所以 k(s) = ∥α′′ (s)∥ = 1。更一般地，半径为 a 的圆的曲率为 1/a。
例 1.4 (直线的曲率). 直线 α(s) = p + vs（v 为单位向量），则 α′′ (s) = 0，所以 k(s) = 0。

注 1.7. 挠率描述曲线在空间中的扭曲程度。平面曲线的挠率恒为零。
例 1.5 (圆柱螺旋线的挠率). 圆柱螺旋线 α(t) = (a cos t, a sin t, bt) 的曲率为 a/(a2 + b2 )，

注 1.8. 曲率是曲线” 弯曲” 的程度，挠率是曲线” 扭曲” 的程度。平面曲线：曲率可能非
零，但挠率恒为零。空间曲线：曲率和挠率都可能非零。

1.5.3    The Frenet-Serret Formulas
Frenet-Serret 公式是微分几何中最重要的公式之一，它建立了曲线的局部不变量之间的关
系。

定义 1.12 (Frenet 标架). 设 α(s) 是单位速度曲线，k(s) > 0。定义：

    • 切向量（tangent vector）：t(s) = α′ (s)

                                           t′ (s)    α′′ (s)
    • 法向量（principal normal）：n(s) =                 =
                                          ∥t′ (s)∥   k(s)

    • 副法向量（binormal）：b(s) = t(s) × n(s)

{t, n, b} 称为 Frenet 标架（Frenet frame）。

注 1.9. {t, n, b} 在每一点构成 R3 的标准正交基。

    • t 指向曲线前进方向

    • n 指向曲线弯曲方向

    • b 垂直于密切平面

定义 1.13 (密切平面). 由 t(s) 和 n(s) 张成的平面称为密切平面（osculating plane）。

注 1.10. Frenet-Serret 公式表明，如果我们知道曲率 k(s) 和挠率 τ (s)，以及初始的 Frenet
标架，就可以唯一确定曲线的形状。这两个函数完全决定了曲线（在局部意义上）。

推论 1.15. 曲率和挠率可以通过以下公式计算：

                             k = ∥t′ ∥,     τ = −⟨b′ , n⟩.

定理 1.16 (曲线存在唯一性). 给定两个连续函数 k(s) > 0 和 τ (s)，以及 R3 中一组正交标
架，存在唯一一条单位速度曲线 α(s)，使得其曲率为 k(s)、挠率为 τ (s)，且 Frenet 标架
为给定的标架。

注 1.11. Frenet-Serret 公式是曲线的” 运动方程”。曲率 k 和挠率 τ 完全决定了曲线的局
部几何性质。这类似于：给定加速度和角速度，就确定了刚体的运动。

1.6    The Local Canonical Form
利用 Frenet 标架，我们可以把曲线在一点附近展开为标准形式。
定理 1.17 (曲线的局部标准形式). 设 α(s) 是单位速度曲线，s0 是曲线上一点，k(s0 ) > 0，
τ (s0 ) ̸= 0。则在 s0 附近，曲线可以展开为
                                        (s − s0 )2         (s − s0 )3
        α(s) = α(s0 ) + (s − s0 )t0 +              k0 n0 +            k0 τ0 b0 + o((s − s0 )3 ),
                                            2                  6
其中 t0 = t(s0 )，k0 = k(s0 )，τ0 = τ (s0 )。
注 1.12. 这个展开式揭示了曲线的几何意义：
  • 线性项：沿切方向 t 的运动
  • 二次项：沿法方向 n 的弯曲（由曲率 k 控制）
  • 三次项：沿副法方向 b 的扭曲（由挠率 τ 控制）


1.7    Global Properties of Plane Curves
前面我们讨论的是曲线的局部性质。现在我们讨论平面曲线的整体（全局）性质。
定理 1.18 (四顶点定理). 设 C 是平面上一条简单闭曲线，曲率恒为正。则 C 上至少有四
个曲率极值点（顶点）。
注 1.13. ” 顶点”（vertex）是指曲率取得局部极值的点。四顶点定理是平面曲线理论中最
著名的整体定理之一。例如，椭圆有四个顶点（长轴和短轴的端点）。
定理 1.19 (Fenchel 定理). 设 α : [0, L] → R3 是一条简单闭曲线。则其全曲率

即曲率沿曲线的积分。
推论 1.21. 任何简单闭曲线的全曲率至少为 2π。
定理 1.22 (Fary-Milnor 定理). 如果 α 是一条简单闭曲线的全曲率小于 4π，则该曲线是未
打结的（unknotted）。
注 1.14. 四顶点定理是平面曲线的整体性质：曲线的弯曲程度必须” 平衡”。Fenchel 定理
说明：闭合曲线的” 总弯曲量” 至少是 2π。这些定理连接了曲线的局部几何（曲率）和全
局拓扑性质。

1.8 本章总结
本章介绍了曲线微分几何的基础概念：

     1. 正则曲线：导数非零的光滑曲线

     2. 弧长：曲线的内在不变量，可用于重新参数化

     3. 曲率 k(s)：描述曲线弯曲程度

     4. 挠率 τ (s)：描述曲线在空间中的扭曲程度

     5. Frenet-Serret 公式：连接 k、τ 与曲线的运动

     6. 局部标准形式：曲线在一点附近的展开

     7. 整体性质：四顶点定理、Fenchel 定理、Fary-Milnor 定理

练习 1.1. 验证圆柱螺旋线 α(t) = (a cos t, a sin t, bt) 的曲率为常数 a/(a2 + b2 )，挠率也为
常数 b/(a2 + b2 )。

练习 1.2. 证明：如果一条曲线的曲率和挠率均为常数，则该曲线或者是圆柱螺旋线，或者
是圆（当挠率为零时）。
