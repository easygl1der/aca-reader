什么是 Vibe Research？

  • Vibe Research = 氛围研究 → 借助 AI 工具营造高效学术研究氛围
  • 核心理念：让 AI 处理繁琐工作，专注核心思考
  • 三大目标：
    • 📚 加快文献阅读 — 自动化提取、总结论文要点
    • 📝 做好学习记录 — 结构化笔记、跨会话记忆
    • 🔄 工作流自动化 — 从 PDF 到 LaTeX 一键生成

传统研究 vs AI 辅助研究

传统方式

  • 手动复制 PDF 内容
  • 逐页阅读，效率低下
  • 笔记分散在不同工具
  • 难以追踪阅读进度
  • 引用格式需要手动维护

AI 辅助方式

  • 自动 OCR + 结构化提取
  • AI 总结 + 关键问题生成
  • 统一 LaTeX 格式笔记
  • 进度自动记录 + 可视化
  • BibTeX 自动生成

LaTeX 笔记自动生成

  • 格式规范：纯 LaTeX，禁止 Markdown
  • 标签系统：\label{def:名称} + \cref{引用}
  • 问答记录：\userannotation 自动追加
  • 跨文件引用：章节间交叉引用

📝 Stein 风格写作：每个概念引入前，先解释"为什么"；强调动机、历史脉络、有机联系。

案例：Schubert 多项式 (1)

  • 主题：Schubert 多项式与量子乘法
  • 目标：整理量子 Schubert 计算的研究笔记
  • 挑战：多篇论文、符号复杂、图示繁多

工作流步骤

  1. PDF 收集：从 arXiv、Google Scholar 下载相关论文
  2. minerU 转录：批量转换为 Markdown
  3. Claude Code 逐篇阅读：提取核心定理、定义、证明思路
  4. LaTeX 笔记整理：按主题组织，加入自己的理解

📌 关键：不是简单复制论文，而是用 Stein 风格 重写——动机驱动、有机关联。

案例：Schubert 多项式 (2)

笔记示例：Schubert 多项式定义

% LaTeX 笔记片段 \section{Schubert 多项式简介} % 动机驱动写作 (Stein 风格) Schubert 多项式的出现源于一个基本问题： 如何计算旗流形 (flag variety) $Fl_n$ 的结构常数？ 与经典情形不同，量子 cohomology $QH^*(Fl_n)$ 中的乘法涉及"3-点函数"， 这对应于Gromov-Witten 不变量。 % 核心定义 \begin{definition}[Schubert 多项式] 对于置换 $w \in S_n$，定义 \[ \mathfrak{S}_w(x) = \sum_{\substack{w = uv \\ u \in S_j}} x_1^{l(u)} \mathfrak{S}_{v}(x_{j+1},\ldots) \] \end{definition}
  • ✅ 自动编号 → \label{def:schubert-poly}
  • ✅ 交叉引用 → \cref{def:schubert-poly}
  • ✅ 问答记录 → \userannotation{...}
