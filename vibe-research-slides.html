什么是 Vibe Research？

  • Vibe Research = 氛围研究 → 借助 AI 工具营造高效学术研究氛围
  • 核心理念：让 AI 处理繁琐工作，专注核心思考
  • 三大目标：
    • 📚 加快文献阅读 — 自动化提取、总结论文要点
    • 📝 做好学习记录 — 结构化笔记、跨会话记忆
    • 🔄 工作流自动化 — 从 PDF 到 LaTeX 一键生成
2 / 15

传统研究 vs AI 辅助研究

传统方式

  • 手动复制 PDF 内容
  • 逐页阅读，效率低下
  • 笔记分散在不同工具
  • 难以追踪阅读进度
  • 引用格式需要手动维护

AI 辅助方式

  • 自动 OCR + 结构化提取
  • AI 总结 + 关键问题生成
  • 统一 LaTeX 格式笔记
  • 进度自动记录 + 可视化
  • BibTeX 自动生成

核心工具链

  • Claude Code — AI 编程助手，理解项目上下文
  • Skill 系统 — 自动化任务封装（20+ 技能）
  • minerU — PDF 转录为 Markdown
  • LaTeX — 标准化学术笔记格式
  • Obsidian — 本地知识管理（可选）

💡 关键洞察：不是用 AI 替代思考，而是用 AI 处理 重复性工作——格式整理、文献查找、进度追踪。

Skill 系统介绍

  • Skill = 封装好的自动化工作流
  • 项目自带的 20+ Skills 覆盖：

文献阅读工作流

加载 PDF → 解析结构 → 逐章阅读 → 实时问答 → 生成笔记
  • 扫描阶段：快速了解内容概览、章节结构
  • 逐章阅读：深入理解，区分核心/细节
  • 交互问答：口语化回答 + 自动记录到笔记
  • Stein 风格：动机驱动、历史脉络、有机关联

LaTeX 笔记自动生成

  • 格式规范：纯 LaTeX，禁止 Markdown
  • 标签系统：\label{def:名称} + \cref{引用}
  • 问答记录：\userannotation 自动追加
  • 跨文件引用：章节间交叉引用

📝 Stein 风格写作：每个概念引入前，先解释"为什么"；强调动机、历史脉络、有机联系。
