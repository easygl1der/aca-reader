# aca-reader

文献阅读与讲义生成工作流

## 概述

一个模拟用户阅读文献、学习知识、生成个性化 LaTeX 讲义的工作流系统。

## 功能

- PDF 文献/教材解析
- 两阶段阅读流程（扫描 + 逐章）
- 交互式问答
- LaTeX 讲义生成（Stein 风格）
- 阅读记忆系统
- 进度仪表盘

## 使用方法

1. 加载 PDF 文件
2. 进行扫描阶段
3. 逐章阅读并提问
4. 查看生成的 LaTeX 讲义
5. 导出 PDF
