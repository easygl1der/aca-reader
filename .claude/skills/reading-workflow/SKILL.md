# 文献阅读主工作流

## 概述

这是主要的工作流入口，协调各个子模块。

## 工作流程

### 阶段一：扫描
1. 加载 PDF
2. 解析章节结构
3. 生成章节概览
4. 识别核心定理/定义

### 阶段二：逐章阅读
1. 用户选择章节
2. 展示章节概览和核心目标
3. 进入交互式问答模式
4. 实时更新讲义

### 阶段三：进度管理
1. 记录阅读位置
2. 更新进度仪表盘
3. 保存问答历史

## 子模块调用

作为 Orchestrator，本 skill 协调以下子模块：

- **pdf-reader**: 加载和解析 PDF 文件
- **reading-progress**: 管理阅读进度和状态
- **latex-notes**: 生成和更新 LaTeX 讲义
- **interactive-qa**: 处理用户问答

当需要时，使用 Agent 工具调用相应的子 skill。

## 使用方式

- "帮我加载这个 PDF"
- "开始扫描"
- "我要读第3章"
- "我的阅读进度如何"
- "生成讲义"
