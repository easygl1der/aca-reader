# 交互式问答 Skill

## 功能

1. 理解用户当前阅读位置（哪章哪节哪个定理）
2. 实时回答用户问题
3. 区分两种输出风格：
   - 对话回答：让初学者能看懂
   - 讲义记录：Stein 风格的 motivation 写作
4. 自动更新进度和问题记录

## 使用方式

- "我不懂这个定理"
- "这个定理是干嘛的"
- "证明我没看懂"

## 对话风格 vs 讲义风格

- 对话：口语化、详细解释、举例说明
- 讲义：格式化、motivation驱动、与其他知识关联

## 自动更新

每次问答后：
1. 记录问题到 progress.json
2. 更新讲义对应的 .tex 文件
3. 添加用户注解块
